How do I install Nivix Studio?

First. You need these 2 programs:
Google Chrome
Node.js
Next
Download the installer from the link above. This will give you a zip file containing a few files. All are required to run. Extract or copy the contents of that zip file to wherever you want to have Nivix Studio installed.
On Windows:
run nivix-studio-installer-windows.bat by clicking on it twice in rapid succession
On Linux:
Assuming you're running gnome (popular for linux desktop environments, default on Fedora and Ubuntu), you have to right click on nivix-studio-installer-linux.sh and click properties, then scroll to the bottom and make sure the "Executable as program" tick is enabled. Then click off of the menu and right click the .sh file and click "Run as a program".


Where can I use Nivix Studio?

Nivix Studio works on many platforms, including:
•Windows 10/11
•Linux RedHat Based Distributions (Like Fedora)
•Linux Debian Distributions (Like Ubuntu)


Can I use Nivix Studio from a USB drive?

Absolutely! You can install Nivix Studio to just about anywhere you want as long as it has more than 25mb of free storage space. BUT in order to run it, you need to plug that USB into a host machine that runs Windows or Linux (Debian/Redhat)


Where does Nivix Studio store data?

In your OS' home directory. Examples:
•C:\Users\(your_user_name)\nvxstdo\ on Windows
•/home/nvxstdo/ on Linux


Can I downgrade Nivix Studio?

Sure! The goal is to keep every version of Nivix Studio as functional as possible. As well as making them all as available as possible (Scroll down to find history of versions). Just beware when doing so, old versions are missing new Security and bug patches.
